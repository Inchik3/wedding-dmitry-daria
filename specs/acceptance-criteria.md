# Acceptance Criteria

## Конверт

- Given гость открывает сайт, When страница загружена, Then сразу виден светлый конверт без отдельного прелоадера.
- Given конверт виден, When гость нажимает «Открыть приглашение», Then конверт закрывается и показывается основная страница.
- Given включен `prefers-reduced-motion`, When гость открывает приглашение, Then используется упрощенный переход без интенсивного движения.

## Музыка

- Given сайт загружен, When гость не нажимал «Открыть приглашение», Then музыка не запускается.
- Given файл `public/audio/wedding-theme.mp3` существует, When гость открывает приглашение, Then музыка может запуститься в результате пользовательского действия.
- Given файл музыки отсутствует, When гость открывает приглашение, Then сайт работает, а кнопка музыки показывает нейтральное состояние без console errors.
- Given музыка играет, When гость нажимает кнопку звука, Then воспроизведение выключается и `aria-pressed` обновляется.

## Hero и фотографии

- Given `hero.webp` доступен, When основная страница открыта, Then hero показывает фото Дмитрия и Дарьи с читаемым текстом.
- Given `hero.webp` отсутствует, When основная страница открыта, Then отображается светлая заглушка с сохраненной композицией.
- Given дополнительные фото отсутствуют, When гость доходит до фотопауз, Then видны заглушки «Здесь будет наша фотография».
- Given дополнительные фото доступны, When гость доходит до фотопауз, Then они загружаются лениво.
- Given проект готов, Then в нем нет сторонних фотографий и фото из референса.

## Таймер

- Given текущий момент раньше 28 августа 2026 13:00 Europe/Moscow, When страница открыта, Then таймер показывает дни, часы, минуты и секунды.
- Given текущий момент равен или позже целевой даты, When страница открыта, Then таймер заменен текстом «Сегодня наш особенный день!».
- Given устройство находится в другом часовом поясе, When таймер рассчитывает остаток, Then результат основан на `2026-08-28T13:00:00+03:00`.

## Программа дня

- Given гость видит программу, Then отображаются только четыре события: 12:45, 13:00, 15:45, 16:00.
- Given программа отображается, Then нет фотосессии, фуршета, первого танца, торта, завершения вечера, церемонии или других неутвержденных событий.

## Места и карты

- Given гость видит места, Then есть регистрация по адресу «г. Саранск, ул. Коммунистическая, 61».
- Given гость видит места, Then есть банкет в Rose park по адресу «г. Саранск, ул. Строительная, 15».
- Given iframe карты не загрузился, When гость смотрит блок места, Then адрес и кнопка «Построить маршрут» остаются доступными.
- Given гость нажимает «Построить маршрут», Then открывается публичная ссылка Яндекс Карт.
- Given координаты не подтверждены, Then реализация не использует выдуманные координаты.

## Дресс-код

- Given секция дресс-кода открыта, Then текст мягко просит пастельную гамму и не звучит как ультиматум.
- Given файл палитры доступен, When проводится финальная настройка, Then основные HEX извлечены из реального изображения и зафиксированы.
- Given файл палитры отсутствует, Then сайт показывает текст/placeholder без выдуманных HEX.

## Навигация и адаптивность

- Given viewport 320px, 390px, 768px, 1024px или 1440px, Then нет горизонтальной прокрутки и incoherent overlap.
- Given mobile viewport, When гость открывает меню, Then пункты доступны и закрываются после выбора.
- Given keyboard navigation, When гость нажимает Tab, Then фокус идет по интерактивным элементам в понятном порядке.

## RSVP validation

- Given форма пустая, When гость отправляет ответ, Then показаны ошибки обязательных полей.
- Given выбран статус «Буду присутствовать» и напитки не выбраны, When гость отправляет форму, Then показана ошибка выбора напитков.
- Given выбран статус «Не смогу присутствовать», When форма отображается, Then блок напитков скрыт и напитки не обязательны.
- Given комментарий длиннее лимита, When гость отправляет форму, Then показана ошибка длины.
- Given сейчас позже 1 августа 2026, When форма отображается, Then показано сообщение «Срок подтверждения уже прошёл, но вы всё ещё можете отправить ответ».
- Given late response отправлен, Then payload содержит `isLate: true`.

## RSVP отправка

- Given Apps Script URL отсутствует, When гость видит RSVP, Then форма сообщает, что пока не подключена, и сайт не ломается.
- Given endpoint отвечает success, When гость отправляет валидную форму, Then показано «Спасибо! Ваш ответ сохранён.».
- Given endpoint отвечает ошибкой или сеть недоступна, When гость отправляет валидную форму, Then показана ошибка, введенные данные сохранены и доступна повторная попытка.
- Given honeypot заполнен, When запрос приходит на сервер, Then Apps Script не записывает его как обычный RSVP.

## Apps Script и таблица

- Given Apps Script получает валидный POST, Then он валидирует данные на сервере и записывает строку в лист `RSVP`.
- Given запись создана, Then таблица содержит `submittedAt`, `guestName`, `attendance`, `drinks`, `comment`, `isLate`, `source`, `requestId`.
- Given запрос обработан, Then User-Agent не записывается.
- Given ошибка сервера, Then клиент получает понятное error-сообщение.

## GitHub Pages

- Given `VITE_BASE_PATH=/repository-name/`, When выполняется production build, Then ссылки на ассеты работают с этим base path.
- Given workflow запускается на push, Then выполняются typecheck, tests и build перед публикацией.
- Given публикация фактически не выполнялась, Then итоговый отчет не утверждает, что сайт опубликован.

## Тесты и build

- Given реализация завершена, When выполняется `npm run typecheck`, Then TypeScript проходит без ошибок.
- Given реализация завершена, When выполняется Vitest, Then unit tests проходят.
- Given реализация завершена, When выполняется Playwright, Then E2E сценарии проходят или явно указано, что проверить не удалось.
- Given реализация завершена, When выполняется `npm run build`, Then build завершается успешно.
- Given страница проверяется в браузере, Then нет необработанных console errors.

## Аналитика и приватность

- Given проект готов, Then в нем нет аналитики.
- Given RSVP отправляется, Then User-Agent не собирается.
- Given клиентский код собран, Then в нем нет секретов.
